- Gerar o ambiente virtual: virtual venv
- Ativar o ambiente virtual: source venv/bin/activate
- Instalar o Django: pip install Django
- pip freeze > requirements.txt (boa prática)
- django-admin help
- django-admin startproject setup .
- python manage.py runserver
- pip install python-dotenv
- pip freeze > requirements.txt
- python manage.py startapp galeria (em um projeto, temos diversos apps)
- Adicionar o app galeria no settings
- Adicionar a pasta templates no settings
- Pegar os arquivos estáticos: python manage.py collectstatic
